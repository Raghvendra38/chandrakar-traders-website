Set Up Contact Form to Receive Emails (Free)

Free — Formspree

Currently the form shows a success message but doesn't actually send you an email. Fix this in 5 minutes using Formspree (free, no server needed):

  1. Go to formspree.io and sign up free
  2. Click New Form, enter your email address
  3. Copy your form endpoint — it looks like: https://formspree.io/f/xabc1234
  4. Open contact.html and find this line in script.js:
// In js/script.js — replace the setTimeout with a real fetch:

form?.addEventListener('submit', async (e) => {
  e.preventDefault();
  const btn = form.querySelector('.btn-submit');
  btn.textContent = 'Sending…';
  btn.disabled = true;

  const data = new FormData(form);

  const response = await fetch('https://formspree.io/f/YOUR_FORM_ID', {
    method: 'POST',
    body: data,
    headers: { 'Accept': 'application/json' }
  });

  if (response.ok) {
    form.reset();
    formSuccess.style.display = 'block';
    formSuccess.textContent = '✓ Message sent! We will contact you soon.';
    btn.textContent = 'Send Message →';
    btn.disabled = false;
  }
});
💡 Free plan: Formspree free plan allows 50 form submissions per month — more than enough to start.

Upload Your Website to GitHub

Required for Deployment

GitHub stores your website files so Cloudflare can publish them.

  1. Go to github.com and create a free account if you don't have one
  2. Click the + button (top right) → New repository
  3. Name it chandrakar-traders-website, set it to Public, click Create repository
  4. Install GitHub Desktop from desktop.github.com (easiest way — no command line needed)
  5. Open GitHub Desktop → Add an Existing Repository → select your website folder
  6. Click Publish repository to push your files to GitHub
⚠️ Important: Make sure your folder has this exact structure before uploading: index.html, about.html, products.html, contact.html, css/style.css, js/script.js, images/ (with all photos), sitemap.xml, robots.txt

Or use the Command Line:

cd your-website-folder

git init
git add .
git commit -m "Initial website upload"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/chandrakar-traders-website.git
git push -u origin main

Deploy FREE on Cloudflare Pages

100% Free — Unlimited Traffic

Cloudflare Pages will host your website for free with fast global CDN, HTTPS and unlimited traffic.

  1. Go to pages.cloudflare.com and create a free account
  2. Click Create a project → Connect to Git
  3. Click Connect GitHub and authorize Cloudflare
  4. Select your repository chandrakar-traders-website
  5. In Build settings: leave everything blank (no build command needed for static HTML)
  6. Click Save and Deploy
  7. Wait 1–2 minutes. Your site will be live at: chandrakar-traders-website.pages.dev
🎉 Auto-deploy: Every time you push new changes to GitHub, Cloudflare automatically updates your live website within 1 minute!

Build Settings (leave as is):

Add a Custom Domain (Optional but Recommended)

~₹800/year

Your site will work at chandrakar-traders.pages.dev for free. For a professional domain like chandrakartraders.in:

  1. Buy a domain from GoDaddy.com or BigRock.in — search for chandrakartraders.in (costs ~₹600-900/year)
  2. In Cloudflare Pages, go to your project → Custom domains → Set up a custom domain
  3. Enter your domain name and follow the DNS instructions
  4. In your domain registrar (GoDaddy/BigRock), update the Nameservers to the ones Cloudflare gives you
  5. Wait 10–30 minutes for DNS to propagate — your site will be live at your own domain!
💡 HTTPS is automatic: Cloudflare provides free SSL certificate — your site will be secure (https://) automatically.

After you get your domain, replace yourdomain.com in these files with your real domain:

  • index.html — in the JSON-LD schema and OG tags
  • sitemap.xml — all 4 URLs
  • robots.txt — the sitemap URL

Add Free Analytics (Track Visitors)

Free

Option A — Cloudflare Web Analytics (Easiest, Free, No Cookies)

  1. In Cloudflare dashboard → Web Analytics → Add a site
  2. Copy the script snippet they give you
  3. Paste it before </body> in ALL 4 HTML files
<!-- Paste before </body> in all pages -->
<script defer src='https://static.cloudflareinsights.com/beacon.min.js'
  data-cf-beacon='{"token": "YOUR_CLOUDFLARE_TOKEN"}'></script>

Option B — Google Analytics (More Detailed)

  1. Go to analytics.google.com → sign in with Google
  2. Create an account → create a Property → choose Web
  3. Enter your website URL, get your Measurement ID (looks like G-XXXXXXXXXX)
  4. Paste this in <head> of ALL 4 HTML files:
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>

Submit to Google Search

Free — Very Important for SEO

Tell Google your website exists so it shows up in search results for "motor pump Amleshwar", "cement supplier Raipur" etc.

  1. Go to search.google.com/search-console
  2. Click Add Property → enter your website URL
  3. Verify ownership — easiest: choose HTML tag method, copy the meta tag and add it to <head> of index.html
  4. Once verified, click Sitemaps → enter sitemap.xml → Submit
  5. Click URL Inspection → enter your URL → Request Indexing
⏱ Timeline: Google usually indexes your site within 3–7 days. After 2–4 weeks, search "chandrakar traders Amleshwar" and you should see your site appear!

Also register on Google Business Profile (business.google.com) to appear on Google Maps when people search "motor pump near me".
